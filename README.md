> Portafolio profesional construido con Astro + TailwindCSS. Diseño oscuro, minimalista y orientado a conversión.

## ✨ Características

- **Diseño oscuro** con acentos azul/esmeralda
- **Mobile-first** responsive
- **Case Studies** expandibles con métricas concretas
- **Stack tecnológico** con iconos Devicon
- **Proyectos** con carrusel de imágenes
- **SEO optimizado** (OpenGraph, JSON-LD)
- **Transiciones suaves** mediante View Transitions API

## 📦 Instalación

```bash
# Clonar el repo
git clone https://github.com/bessondev/portafolio-bessondev.git
cd portafolio-bessondev

# Instalar dependencias
npm install

# Desarrollo local
npm run dev

# Build para producción
npm run build
```
